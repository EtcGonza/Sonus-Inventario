# Sistema de Inventario — Micrófonos de Guitarra

Este repositorio contiene la documentación inicial para un sistema pensado para un microemprendimiento que arma micrófonos de guitarra a mano.

# Documentación del proyecto — índice principal

Hemos movido la documentación informativa (archivos .md raíz) a la carpeta `docs/` para mantener el repositorio más organizado.

- Documentación informativa (moved): see `docs/` — contiene:

  - `docs/OBJETIVOS.md`
  - `docs/ALCANCES.md`
  - `docs/MATERIALES.md`
  - `docs/FUNCIONALIDADES.md`
  - `docs/PRODUCTOS.md`
  - `docs/RECETAS.md`

- Documentación técnica: `technical/` — modelos, interfaces, servicios y guías de implementación.

Cómo empezar

1. Leer `docs/README.md` para navegar por la documentación informativa.
2. Revisar `technical/SETUP.md` para pasos de instalación, dependencias y ESLint mínimo.
3. Revisar `technical/project_structure.md` para la estructura propuesta del frontend (Angular 21, standalone).
