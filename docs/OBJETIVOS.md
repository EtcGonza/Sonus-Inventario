# Objetivos del proyecto

1. Registrar y mantener actualizado el stock de materiales necesarios para armar micrófonos de guitarra.
2. Facilitar la gestión básica de entradas y salidas de materiales (cantidad disponible).
3. Proveer un módulo que calcule y muestre qué micrófonos se pueden armar con el stock actual.
4. Mantener la documentación clara y suficiente para iterar hacia una versión mínima viable (MVP).

Notas:

- El enfoque inicial es funcionalidad mínima y datos simples para permitir una puesta en marcha rápida.
