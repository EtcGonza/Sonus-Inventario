# Alcances iniciales

Incluido en la primera versión (MVP):

- Registro de materiales con información mínima.
- Interfaz (puede ser CLI, web o móvil) para listar y actualizar cantidades de materiales.
- Definición de recetas/BOMs por tipo de micrófono (lista de materiales y cantidades requeridas).
- Módulo "¿Qué puedo armar?" que calcula para cada tipo de micrófono cuántas unidades se pueden construir con el stock actual.

No incluido en la primera versión (posible roadmap):

- Gestión avanzada de proveedores y órdenes de compra.
- Control de lotes, caducidades o trazabilidad compleja.
- Usuarios/roles con permisos granulares.
- Integración con contabilidad o e-commerce.
